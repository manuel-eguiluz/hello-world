# hello-world-expressjs-ts

This is a basic hello world using:

- ExpressJS
- TypeScript
- Nodemon

## How to use

Run the application.

```(shell)
npm run build       #Build the app
npm run start       #Run the app

or

npm run start:dev   #Run the app in dev mode
```

Get the message.

```(shell)
curl http://localhost:3000
```
